=================================
How to use 0-add_integer
=================================

This module provides a function `add_integer(a, b=98)` that adds two integers.

First, import the function:
    >>> add_integer = __import__('0-add_integer').add_integer

    >>> add_integer(1, 2)
    3
    >>> add_integer(100, -2)
    98
    >>> add_integer(2)
    100
