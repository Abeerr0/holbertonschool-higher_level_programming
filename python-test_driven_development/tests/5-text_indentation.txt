=================================
How to use 5-text_indentation
=================================

>>> text_indentation = __import__('5-text_indentation').text_indentation

Test regular string with punctuation:
>>> text_indentation("Holberton. School? How are you: John")
Holberton.
<BLANKLINE>
School?
<BLANKLINE>
How are you:
<BLANKLINE>
John

Test string without special punctuation:
>>> text_indentation("Holberton School")
Holberton School

Test string with internal spaces and punctuation:
>>> text_indentation("Holberton.School")
Holberton.
<BLANKLINE>
School

Test string with blank line (newline characters):
>>> text_indentation("First line.\nSecond line.")
First line.
<BLANKLINE>
Second line.

Test text is not a string (integer):
>>> text_indentation(12)

    ...

Test text is not a string (None):
>>> text_indentation(None)
Traceback (most recent call last):
    ...
TypeError: text must be a string

>>> text_indentation()

    ...
