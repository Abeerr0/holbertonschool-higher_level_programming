=====================================
How to use 5-text_indentation module
=====================================

>>> text_indentation = __import__('5-text_indentation').text_indentation

Test normal string:
>>> text_indentation("Holberton School")
Holberton School

Test dots:
>>> text_indentation("Holberton.School")
Holberton.
<BLANKLINE>
School

Test mix:
>>> text_indentation("Holberton. School? How are you: John")
Holberton.
<BLANKLINE>
School?
<BLANKLINE>
How are you:
<BLANKLINE>
John

Test not a string:
>>> text_indentation(12)
Traceback (most recent call last):
    ...
TypeError: text must be a string

Test multiple lines as requested by checker:
>>> text_indentation("First line.\nSecond line.")
First line.
<BLANKLINE>
Second line.
